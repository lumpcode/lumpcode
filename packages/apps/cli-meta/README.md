# lumpcode

Alias for [Lumpcode CLI](https://www.npmjs.com/package/@lumpcode/cli).

Unscoped npm package that installs the [Lumpcode CLI](https://www.npmjs.com/package/@lumpcode/cli).

```bash
npm install -g lumpcode
```

This package depends on `@lumpcode/cli` and forwards the `lumpcode` command. Documentation: [github.com/lumpcode/lumpcode](https://github.com/lumpcode/lumpcode).
